Semantic Vega v1.19 preview-tab hotfix

- Preview/inspector tab JavaScript syntax errors caused by raw </script> sequences inside HTML strings embedded in an inline parent <script>.

- Web package filename insertion now uses JSON.stringify(filename), so filenames are emitted as valid JavaScript strings.
